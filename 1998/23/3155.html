Nmódosított = (Nátlagos+Nátlagos * p) * q [db],
2 * (Anyert átlagos – Aajánlati)
ahol p = de –0,5 < p < +0,5,
Anyert átlag
0,9 < q < 1,1 (a két részteljesítésre összesen = 2)
Az algoritmus változói:
Nmódosított = módosított szállítási mennyiség;
Nátlagos = átlagos szállítási mennyiség;
p = a mennyiség-eltérítés ár összetevõje;
q = a mennyiség-eltérítés mûszaki tartalom összetevõje;
Anyert átlag = nyertes szállítók átlagára;
Aajánlati = az adott nyertes szállító ajánlati ára.
